    serial parity <none/odd/even>

    Description

    Configures the parity bit to none, odd, or even.

    Example

    [admin@ssh-to-serial]>serial parity odd
UART Config - Base: 0x4000D000
 Clock: 120000000 Hz
 Baud: 115200
 Data Size: 8
 Parity: O
 Stop Bits: 1
 Flow Control: N
 Transmit Delay: 100 ms
Serial parity set to O.
