# `serial stopbits <bits>`

## Description 
Configures the number of stop bits (either `1` or `2`).

## Example 
```
[admin@ssh-to-serial]>serial stopbits 1
UART Config - Base: 0x4000D000
 Clock: 120000000 Hz
 Baud: 9600
 Data Size: 5
 Parity: N
 Stop Bits: 1
 Flow Control: N
 Transmit Delay: 100 ms
Serial stop bits set to 1.
```S
