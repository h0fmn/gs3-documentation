# `serial datasize <bits>`

## Description
Configures the data size, with a range of `5` to `8` bits. 

## Example 
```
[admin@ssh-to-serial]>serial datasize 5
UART Config - Base: 0x4000D000
 Clock: 120000000 Hz
 Baud: 9600
 Data Size: 5
 Parity: N
 Stop Bits: 1
 Flow Control: N
 Transmit Delay: 100 ms
Serial data size set to 5 bits.
```
