# `serial show`

#### Description
Displays the current serial configuration.

#### Example
```
[admin@ssh-to-serial]>serial show
UART Config - Base：0x4000D000
 Clock：120000000 Hz
 Baud：115200
 Data Size：8
 Parity：N
 Stop Bits：1
 Flow Control：N
 Transmit Delay：0 ms
Current configuration displayed.
```
