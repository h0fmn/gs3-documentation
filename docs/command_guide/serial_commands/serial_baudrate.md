# `serial baudrate`

## Description
Configures the baud rate. The default is set to `115200`.

## Example
```
[admin@ssh -to -serial]>serial baudrate 9600
UART Config - Base: 0x4000D000
Clock: 120000000 Hz
Baud: 9600
Data Size: 8
Parity: N
Stop Bits: 1
Flow Control: N
Transmit Delay: 100 ms
Serial baud rate set to 9600.
```
